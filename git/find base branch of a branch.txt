Git: find base branch of a branch
to get the base brach of your feature-branch
==========================================

git clone the <branch_repo>
cd repo_directory
$git show-branch -a \
	| grep '\*' \
	| grep -v `git rev-parse --abbrev-ref HEAD` \
	| head -n1 \
	| sed 's/.*\[\(.*\)\].*/\1/' \
